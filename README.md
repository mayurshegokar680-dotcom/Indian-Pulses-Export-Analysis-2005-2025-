# India_Pulses_Export_Analysis.pbix
This project analyzes the export performance of pulses from India over a two-decade period (2005–2025) using Microsoft Power BI.  The objective was to evaluate export growth, variability, instability, and trend reliability to derive actionable business insights.
# Export performance and growth of pulses from India – A 20 Year Analysis.

## Project Overview

This project analyzes the **export performance and growth of pulses from India** using Power BI.
The dashboard provides insights into export trends, growth patterns, and country-wise export distribution.

## Objectives

* To Analyze growth trends in pulses exports
* To Identify top export destinations
* To Visualize export performance over time
* To Support trade and agriculture market insights

## Tools Used

* Power BI
* Microsoft Excel
* Data Visualization
* Data Analysis

## Key Insights

* Export growth trends across years
* Major countries importing pulses from India
* Share of different pulse categories in exports
* Year-wise export performance analysis

## How to Use

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Explore the interactive dashboard
